How does JavaScript work?

Answer:

JavaScript is a high level programming language and we can easily write this code. Because this code is text type that means human readable.

So, the question is how does JavaScript work?

When we write the codes it will be human readable and machine or computer didn't understand the codes directly. Machine can understand the codes and commands when it is compiled or executed. JavaScript make the codes human readable to machine language. JavaScript is just in time compiled language. It compile all the codes human readable to machine language within some milliseconds. So the machine can understand the language or commands. Also, JavaScript make our website dynamic and interactive. JavaScript can access and make changes to any DOM element.

What does JavaScript event loop do?

Answer:

We know that JavaScript is single threaded programming language. So it will run only one task at a time. By using the setTimeout we can set the execution time for some specific code or function.

Now the question is what does JavaScript event loop do?

JavaScript event loop will create a call stack, api and que. First, the codes will go to the call stack. Call stack will determine which call will take some time and gather them into the que list and run the next called function. When the browser api give response and the queued lists program or command are executed successfully then it will ready to show the output to the browser. The queued function will move to the call stack after completing their api response and execution. Then the program will show the output.
